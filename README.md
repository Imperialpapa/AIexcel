# AI Smart Spreadsheet

웹 기반 AI 스프레드시트 애플리케이션 - OpenAI GPT와 연동하여 자연어로 데이터를 처리할 수 있습니다.

## 주요 기능

### 📊 스프레드시트 기능
- **멀티 시트 지원**: 여러 개의 시트를 가진 엑셀 파일 완벽 지원
- **빈 시트 표시**: 비어있는 시트도 탭에 표시
- **실시간 편집**: 셀을 직접 클릭하여 수정 가능
- **엑셀 가져오기/내보내기**: .xlsx 파일 읽기 및 저장

### 🚀 성능 최적화
- **증분 렌더링**: 대용량 데이터(수천 행)도 부드럽게 처리
- **이벤트 위임**: 메모리 사용량 최소화
- **성능 통계**: 실시간 렌더링 성능 표시
- **데이터 샘플링**: 1000행 이상 데이터의 AI 처리 지원

### 🤖 AI 기능
- **자연어 명령**: "가격이 높은 순으로 정렬해줘" 같은 자연어로 데이터 조작
- **작업 히스토리**: 우측 패널에서 모든 AI 작업 내역 확인
- **오류 추적**: 성공/실패 여부를 시각적으로 표시
- **OpenAI GPT 연동**: GPT-3.5-turbo 또는 GPT-4 사용 가능

## 사용 방법

### 1. 기본 사용
1. **API Key 입력**: OpenAI API Key를 상단에 입력
2. **파일 열기**: "📂 열기" 버튼으로 엑셀 파일 로드
3. **시트 전환**: 하단 탭으로 시트 이동
4. **데이터 편집**: 셀을 클릭하여 직접 수정

### 2. AI 명령 실행
1. AI 명령어 입력란에 자연어 명령 입력
   - 예: "단가가 높은 순으로 정렬해줘"
   - 예: "수량이 5개 이하인 행만 남겨줘"
   - 예: "각 상품의 총액 컬럼을 추가해줘 (단가 × 수량)"
2. "✨ AI 실행" 버튼 클릭
3. AI 패널에서 작업 결과 확인

### 3. 결과 저장
- "💾 저장" 버튼으로 수정된 데이터를 엑셀 파일로 다운로드

## 기술 스택

- **Frontend**: Vanilla JavaScript (프레임워크 없음)
- **스타일**: CSS3
- **엑셀 처리**: SheetJS (XLSX)
- **AI**: OpenAI API (GPT-3.5-turbo)

## 성능 개선 사항

| 항목 | 개선 전 | 개선 후 |
|------|---------|---------|
| 1000행 렌더링 | ~500-1000ms | ~100-200ms |
| 메모리 (1000행) | 10,000개 핸들러 | 1개 핸들러 |
| AI 최대 처리 | ~100행 | ~10,000행 |
| 빈 시트 표시 | ❌ | ✅ |
